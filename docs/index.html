Features

Everything you need for a password manager, in one embeddable crate.

🔐

Authenticated encryption

XChaCha20-Poly1305 over a per-vault data key, wrapped by an Argon2id-derived key. Unique nonce per value, built-in tamper detection.

🗃️

Encrypted SQLite storage

Reliable, ACID-compliant storage with hierarchical folders, soft-delete and full undo.

💾

Backup & restore

Versioned ZIP backups with integrity verification and automatic cleanup of old copies.

🔎

Search

Full-text search across item names and field values, with match highlighting.

🎲

Password generation

Random, pattern-based and memorable password generators built in.

🌍

11 languages

Localized labels and metadata embedded at compile time, plus 20 built-in field types.

Security

Modern, well-reviewed cryptography, no home-grown schemes.

Per-vault data key

A random 256-bit Data Encryption Key protects all data. It is stored wrapped under a key derived from your master password with Argon2id (memory-hard, per-vault salt).

Tamper detection

Every value is sealed with a unique nonce and an authentication tag, so identical secrets never look alike and any modification is caught on decryption.

Instant password changes

Changing the master password only re-wraps the data key. Your stored data is never re-encrypted.

Keys zeroized in memory

Sensitive key material is wiped from memory when the wallet is locked or dropped.

Open source by design

The security of a password manager should be verifiable, not taken on trust.

  • Auditable. Anyone can read exactly how vaults are encrypted, how keys are derived and how data is stored. Security through transparency, not obscurity.
  • Reusable. iwcore is a standalone, MIT-licensed library. Use it to build your own password manager or an alternative client to IntelliWallet.
  • Improvable. Anyone can propose enhancements, open a pull request or fork the project. The library grows with its community.
  • Yours for the long run. Because the storage format and cryptography are open, you can always recover and use your data independently, regardless of the future of the IntelliWallet app or IntelliSoftAlpin eG. Your vault outlives any single product or company.
  • Standard cryptography. Built on well-reviewed primitives from the Rust community (XChaCha20-Poly1305, Argon2id).

Quick start

Add iwcore = "0.2.2" to your Cargo.toml and you are ready.

use iwcore::Wallet;
use std::path::Path;

// Create a vault, add an item and some fields
let mut wallet = Wallet::create(Path::new("./my_wallet"), "my_password", "en")?;
let item = wallet.add_item("My Email", "document", false, None)?;
wallet.add_field(&item, "MAIL", "user@example.com", None)?;
wallet.add_field(&item, "PASS", "secret123", None)?;

// Search
for result in wallet.search("email")? {
    println!("Found: {}", result.item.name);
}
